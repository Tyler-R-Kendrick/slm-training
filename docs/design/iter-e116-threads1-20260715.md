# E116 single-thread resource diagnostic (2026-07-15)

E116 retried the small E91-style recipe with `OMP_NUM_THREADS=1` and
`MKL_NUM_THREADS=1`, batch size `2`, and a 64-step target. The process again
terminated before writing a checkpoint or summary, at step `22`. Partial
metrics telemetry is durable in
`outputs/runs/iter-e116-threads1-20260715/e116_threads1/metrics.jsonl`.

The shell wrapper did not receive a normal exit-status line, indicating an
external termination rather than a handled training exception. No evaluation
or quality claim is made.

Decision: thread count is not the cause. Keep larger/frozen recipes out of the
active loop until the execution environment exposes termination diagnostics.
